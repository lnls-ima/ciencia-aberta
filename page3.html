Etapas de Produção dos Ímãs do Sirius:

Projeto Mecânico

O projeto mecânico dos ímãs compreende:

  • Desenho de cada componente
  • Especificação das tolerâncias mecânicas
  • Definição dos procedimentos de montagem

Cada vez que o projeto do ímã é alterado as simulações magnéticas precisam ser refeitas para verificar se o campo magnético ainda atende as especificações.
