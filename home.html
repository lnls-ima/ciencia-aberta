Etapas de Produção dos Ímãs do Sirius:

1. Espeficicação

Inicialmente são especificadas a força e a quantidade de cada tipo de ímã do acelerador.

Clique Aqui Para Começar!

2. Simulação Magnética

Em seguida são feitas simulações do campo magnético para projetar um ímã que satisfaça as especificações.

3. Projeto Mecânico

Nesta etapa são detalhados os procedimentos e as peças necessárias para produção do ímã.

4. Fabricação do Protótipo

O projeto mecânico é utilizado para a construção de protótipos.

5. Aprovação do Protótipo

Os protótipos passam por diversos testes para validar o procedimento de fabricação.

6. Lote de Produção

Após a aprovação do protótipo é liberada a fabricação de todos os ímãs.

7. Caracterização Magnética

Antes da instalação, são feitas medidas do campo magnético de cada ímã.

8. Pronto Para Instalação!

Após a caracterização magnética os ímãs já podem ser instalados no acelerador.
